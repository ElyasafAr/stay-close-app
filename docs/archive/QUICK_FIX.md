# פתרון מהיר - האתר לא נטען

## הבעיה
הדף לא נטען בכלל ואין לוגים.

## מה לבדוק:

### 1. בדוק את הטרמינל של Next.js
- האם אתה רואה הודעות כמו "Ready in X ms"?
- האם יש שגיאות אדומות?

### 2. בדוק את הדפדפן
- פתח Developer Tools (F12)
- בדוק את הלשונית Console - האם יש שגיאות?
- בדוק את הלשונית Network - האם קבצים נטענים?

### 3. נסה דף פשוט
נסה לפתוח: `http://localhost:3002/test`
זה דף פשוט מאוד שיעזור לבדוק אם Next.js עובד בכלל.

### 4. אם כלום לא עובד:

```bash
# עצור את השרת (Ctrl+C)
# מחק הכל
Remove-Item -Recurse -Force .next
# הפעל מחדש
npm run dev
```

### 5. בדוק את הפורט
- האם השרת רץ על פורט 3002?
- נסה לפתוח `http://localhost:3000` או `http://localhost:3001`

## מה עשיתי:
- יצרתי דף בדיקה פשוט: `/test`
- הסרתי את Header מה-layout
- הסרתי את useTranslation מהדף הראשי

נסה לרענן את הדף ולראות מה קורה.

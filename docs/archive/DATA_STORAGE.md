- נתונים נשמרים תמיד
- מתאים לאפליקציות גדולות
- כבר מוגדר ב-docker-compose.yml

### אפשרות 3: SQLite (בינוני)
- קובץ database מקומי
- לא דורש התקנה נוספת
- מתאים לפרויקטים בינוניים

# 🔑 קבלת מפתח GROQ API

## ⚠️ הבעיה:

המפתח שלך מתחיל ב-`xai-` - זה מפתח של **xAI**, לא **GROQ**!

GROQ דורש מפתח שמתחיל ב-`gsk_` או `sk-`.

---

## ✅ פתרון: קבל מפתח GROQ

### שלב 1: היכנס ל-GROQ Console

1. פתח: **https://console.groq.com/**
2. היכנס לחשבון שלך (או צור חשבון חדש - **חינמי!**)

### שלב 2: צור מפתח API

1. לחץ על **"API Keys"** בתפריט
2. לחץ על **"Create API Key"**
3. תן שם למפתח (למשל: "Stay Close App")
4. לחץ **"Create"**
5. **העתק את המפתח מיד!** (תוכל לראות אותו רק פעם אחת)

### שלב 3: עדכן את קובץ `.env`

1. פתח את `backend/.env`
2. החלף את השורה:
   ```env
   GROQ_API_KEY=xai-xxxxx    # ❌ מפתח xAI
   ```
   
   ב:
   ```env
   GROQ_API_KEY=gsk_xxxxx     # ✅ מפתח GROQ
   ```

3. **ודא שאין רווחים!**

### שלב 4: הפעל מחדש את השרת

```bash
# עצור את השרת (Ctrl+C)
# הפעל מחדש
cd backend
python3 main.py
```

---

## 🔍 איך לזהות מפתח GROQ?

מפתח GROQ צריך להיראות כך:
- `gsk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
- או: `sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

**לא:**
- `xai-xxxxx` ← זה מפתח xAI
- `sk-xxxxx` (של OpenAI) ← זה מפתח OpenAI

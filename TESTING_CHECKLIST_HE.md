# צ'קליסט בדיקות - Stay Close (גרסה 1.9.3+)

מסמך זה נועד לבדיקה מהירה של תקינות המערכת לאחר כל עדכון גרסה.

## 🖥️ מחשב שולחני (Desktop Web)

### ניווט וכותרת (Header)
- [ ] לוגו: לחיצה מובילה לדף הבית (`/messages`).
- [ ] קישורי ניווט: מעבר תקין בין הודעות, אנשי קשר, הגדרות, צור קשר ואודות.
- [ ] כפתור ניהול: מופיע (רק למנהל) עם האייקון 🛠️ ומוביל לדף הניהול.
- [ ] תפריט משתמש (פינה ימנית/שמאלית):
    - [ ] לחיצה פותחת דרופ-דאון עם שם המשתמש ואימייל.
    - [ ] מוצגת גרסת האפליקציה הנכונה (לדוגמה: 1.8.9).
    - [ ] כפתור התנתקות עובד ומחזיר למסך התחברות.

### הגדרות (Settings)
- [ ] שינוי שפה (עברית/אנגלית):
    - [ ] בחירה בקומבו-בוקס **לא** משנה את השפה מיד.
    - [ ] רק לחיצה על "שמור" מעדכנת את השפה ומרעננת את הדף.
    - [ ] באנגלית: כל האתר מתהפך לשמאל (LTR). בעברית: ימין (RTL).
- [ ] שינוי עיצוב (Theme):
    - [ ] בחירה בבהיר/כהה/אוטומטי נשמרת רק לאחר לחיצה על "שמור".
    - [ ] מצב כהה: טקסט קריא, רקעים כהים, ניגודיות טובה בכפתורים.

### הודעות (Messages)
- [ ] בחירת איש קשר: רשימה נפתחת וניתן לבחור.
- [ ] יצירת הודעה: כפתור "צור הודעה" עובד ומחזיר טקסט מה-AI.
- [ ] העתקה ושיתוף: כפתורי העתקה ושיתוף (ווב) עובדים.

---

## 📱 דפדפן סלולרי (Mobile Web)

### תפריט המבורגר
- [ ] פתיחה/סגירה: לחיצה על ה-X או על ה-Overlay (השטח הכהה) סוגרת את התפריט.
- [ ] אייקונים: לכל קישור בתפריט יש אייקון תואם.
- [ ] עיצוב כפתור יציאה: כפתור גדול וצבעוני בתחתית התפריט.
- [ ] ניווט: לחיצה על קישור סוגרת את התפריט ומעבירה לדף הנכון.

### תצוגה רספונסיבית
- [ ] אין גלילה אופקית (Horizontal Scroll).
- [ ] כרטיסי אנשי קשר: במידה והטקסט באנגלית ארוך, הכפתורים יורדים שורה בצורה מסודרת (`Wrap`).
- [ ] באנר שימוש: מופיע בצורה תקינה ולא חוסם כפתורים.

---

## 🤖 אפליקציית אנדרואיד (Native App)

### פונקציות נייטיב
- [ ] שיתוף נייטיב: לחיצה על שיתוף בהודעה פותחת את תפריט השיתוף של אנדרואיד (ולא של הדפדפן).
- [ ] התראות מקומיות:
    - [ ] הגדרת תזכורת לאיש קשר נשמרת.
    - [ ] קבלת התראה בשעה היעודה (בדיקה עם תזכורת לעוד דקה).
    - [ ] ביטול תזכורת מסיר אותה ממערכת ההתראות של המכשיר.

### כללי (Android)
- [ ] כפתור "חזור" של המכשיר: מתנהג כמצופה (סוגר תפריטים או חוזר דף).
- [ ] מקלדת: לא מסתירה שדות קלט קריטיים (כמו בתיאור איש קשר).

---

## 🛠️ בדיקות קצה (Edge Cases)
- [ ] התחברות עם גוגל: עובד תקין.
- [ ] דף תשלום (Paywall): מופיע כשמנסים להוסיף מעבר ל-5 אנשי קשר (בגרסת החינם).
- [ ] מצב אופליין: האפליקציה מציגה הודעה או נטענת מה-Cache (Service Worker).

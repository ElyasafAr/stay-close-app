# יצירת אייקונים לאפליקציה

הקובץ `icon-192x192.png` ו-`icon-512x512.png` חסרים.

## איך ליצור את האייקונים:

### אפשרות 1: יצירה ידנית
1. צור תמונה בגודל 512x512 פיקסלים
2. שמור אותה כ-PNG
3. העתק ל-`public/icon-512x512.png`
4. צור עותק בגודל 192x192 ושמור כ-`public/icon-192x192.png`

### אפשרות 2: שימוש בכלי אונליין
1. לך ל-https://realfavicongenerator.net/
2. העלה תמונה (לוגו או אייקון)
3. הורד את האייקונים
4. העתק את `icon-192x192.png` ו-`icon-512x512.png` ל-`public/`

### אפשרות 3: יצירה מ-SVG
אם יש לך SVG, השתמש ב-ImageMagick או כלי אחר להמרה:
```bash
convert logo.svg -resize 512x512 public/icon-512x512.png
convert logo.svg -resize 192x192 public/icon-192x192.png
```

### אפשרות 4: אייקון זמני (placeholder)
עד שיהיה אייקון אמיתי, אפשר ליצור אייקון פשוט עם טקסט או צבע.
